## 技术栈

`Vite` + `Vue3.x` + `TypeScript` + `Less` + `React` + `Uni-App` +  `Node` + `Express` + `Sequelize` + `MySql`

## 项目结构

- `admin`: `vite + react` 控制台
- `mobile`: `vite + uni-app` 包含 APP, H5, 微信小程序
- `server`: `vite + node` 服务端，包含控制台和 APP 的接口
