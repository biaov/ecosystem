## 项目结构

- `admin`: `vite + react` 控制台
- `app`: `vite + uni-app`，包含 app, h5, 微信小程序
- `server`: `vite + node` 服务端，包含控制台和 app 的接口

## 项目启动

### 下载项目

```sh

```
