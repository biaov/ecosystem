# ecosystem-admin

生态系统的控制台

## 安装依赖

```sh
npm i
```

## 运行项目

```sh
npm start
```

## 打包项目

```sh
npm run build
```

### 命名规范

- 短横线命名(kebab-case): 文件名, class 类命名, 自定义属性传参
- 帕斯卡命名(PascalCase, 大驼峰命名: CamelCase): 枚举命名, 类型命名
- 小驼峰命名(camelCase): 变量命名, HTML id 属性命名, 自定义属性定参

### 技术栈

- `Vite` + `TypeScript` + `Vue3.x` + `Less` + `Axios`
