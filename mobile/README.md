# 商城 APP/H5/微信小程序

## 运行

### 安装依赖

```sh
npm i
```

### 运行

```sh
npm start
```

## 打包

```sh
npm run build
```

## 部署

- 这里使用的是 docker + nginx 部署
- 参考文章：[docker + nginx + pm2 部署前端项目和后端（nodejs）项目](https://wordpress.biaov.cn/docs/38.html)
